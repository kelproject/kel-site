Get Started

Kel™ is an open-source Platform as a Service (PaaS) from Eldarion® that makes it easy to manage web application deployment and hosting through the entire software lifecycle.

Kel helps DevOps professionals manage their application infrastructure through a layer of tools and components that make Kubernetes accessible and easier to use. Kel builds on Eldarion's 7+ years experience running Gondor, one of the leading Python / Django PaaS solutions.

Components

Kel is built on Kubernetes with components written in Python and Go. The following components are a good entryway to understanding the Kel project:

kel-api

RESTful API built with Django / Pinax, adhering to the JSON:API specification

kel-router

HTTP / TLS reverse proxy

kel-identity

Centralized authentication / authorization for Kel

kel

Kel end-user client command-line tool

kelctl

Kel cluster admin command-line tool

kel-cluster

Kel cluster management Python library
